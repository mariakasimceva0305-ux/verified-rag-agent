# Password Reset Policy

Employees must reset passwords through the self-service portal.

- Standard accounts: reset via email + MFA confirmation.
- Privileged accounts: reset requires Service Desk approval and MFA.
- Emergency reset by Service Desk is allowed only for locked accounts.
- Temporary reset links expire after 30 minutes.
- After reset, all active sessions are revoked.

If a user cannot pass MFA, identity must be verified by Service Desk using two independent factors (employee ID and manager confirmation).
